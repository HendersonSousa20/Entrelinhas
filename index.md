# 🌿 Entrelinhas | Psicologia & Presença

Uma *landing page* de alta conversão para clínica de psicologia, focada em **Psicologia Existencial** e **Fenomenologia**. O design foi concebido para transmitir calma, utilizando princípios de minimalismo orgânico e micro-interações que respeitam o tempo do usuário.

-----

## 💎 Diferenciais do Projeto

  * **Design Sensorial:** Uso de "grainy noise" e animações de respiração (*breath-element*) para reduzir a ansiedade do usuário durante a navegação.
  * **Performance:** Carregamento ultra-rápido utilizando Tailwind via CDN (otimizável para PostCSS) e fontes variáveis do Google Fonts.
  * **UX de Acolhimento:** Sistema de "Check-in Emocional" interativo para engajamento imediato.
  * **Mobile First:** Menu hambúrguer customizado e tipografia responsiva adaptada para leitura em telas pequenas.

-----

## 🛠️ Tecnologias Utilizadas

  * **HTML5 & CSS3:** Semântica avançada para SEO.
  * **Tailwind CSS:** Framework utilitário para estilização rápida e consistente.
  * **JavaScript (ES6+):** \* *Intersection Observer API* para revelação de conteúdo ao rolar (Scroll Reveal).
      * Gerenciamento de estados para Modais de leitura.
  * **Google Fonts:** Combinação das fontes *Instrument Serif* (elegância) e *Inter* (legibilidade).

-----

## 📖 Estrutura de Conteúdo

O site está dividido em seções estratégicas de conversão:

1.  **Hero:** Proposta de valor clara com call-to-action imediato.
2.  **Processo:** Explicação da abordagem clínica (Fenomenologia).
3.  **Atuação:** Áreas de especialidade (Ansiedade, Burnout, Identidade).
4.  **Journal:** Seção de educação em saúde com artigos modais.
5.  **Interactive Check-in:** Ferramenta de autodiagnóstico momentâneo.
6.  **FAQ:** Quebra de objeções (Planos de saúde, reembolso, online).

2.  Entre na pasta:

3.  Abra o arquivo `index.html` no seu navegador ou utilize a extensão **Live Server** do VS Code para Hot Reload.
